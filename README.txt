vcalc - C++ single-threaded TCP server and client (port of previous C project)
Build and run tests:
    make test
Build:
    make

Run server:
    ./server -d users.txt -l server.log -p 33333

Run default client:
    ./client_float -H SHA256 -S c
Make Doxygen documentation
    make doc
View documentation
    make view-doc
Make PDF documentation 
    make pdf-doc
View PDF-documentation
    make view-pdf
